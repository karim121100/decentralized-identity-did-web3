# Decentralized Identity (DID) System (Web3)

This project simulates a basic decentralized identity system.

It helps beginners understand how users can own and manage their identity on blockchain.

## What this project does

- Creates user identities
- Stores DID records
- Verifies user identity
- Shows identity list

## Why this project is useful

You will learn:
- What decentralized identity means
- How DID records work
- How identity verification is done
- How Web3 apps manage user profiles

## Tech used

- JavaScript
- Node.js
- DID concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `identity.js` → DID logic  
- `verify.js` → Verification logic  
- `storage.js` → Data storage  
- `user.js` → User validation  
- `config.js` → Settings  
- `utils.js` → Helpers  
- `logger.js` → Logs  
- `sample.env` → Config example  
- `package.json` → Project info

## Notes

This project is for learning only.  
Do not use this for real identity systems.
